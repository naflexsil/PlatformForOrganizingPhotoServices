## Спецификация API

### Авторизация и профиль

| Метод    | Путь                              | Описание                                                        |
| :------- | :-------------------------------- | :-------------------------------------------------------------- |
| `GET`    | `/api/auth/vk`                    | Вход через VK ID                                                |
| `GET`    | `/api/auth/mock-login`            | Быстрый вход для тестов (?role=PHOTOGRAPHER/USER)               |
| `GET`    | `/api/auth/me`                    | Получение данных текущей сессии                                 |
| `POST`   | `/api/auth/complete-registration` | Завершение регистрации (тег, роль)                              |
| `DELETE` | `/api/users/me`                   | Мягкое удаление аккаунта (скрывает профиль из поиска и списков) |
| `PATCH`  | `/api/users/me/restore`           | Восстановление доступа к аккаунту                               |

### Посты и контент

| Метод    | Путь                      | Описание                                            |
| :------- | :------------------------ | :-------------------------------------------------- |
| `GET`    | `/api/posts`              | Лента (сначала закрепленные)                        |
| `GET`    | `/api/posts/:id`          | Детальная информация о конкретном посте             |
| `POST`   | `/api/posts`              | Создание нового поста (JSON: description, images[]) |
| `PUT`    | `/api/posts/:id`          | Редактирование поста (только для автора)            |
| `PATCH`  | `/api/posts/:id/pin`      | Закрепить/открепить                                 |
| `DELETE` | `/api/posts/:id`          | Полное удаление поста из базы                       |
| `POST`   | `/api/posts/:id/like`     | Поставить/убрать лайк                               |
| `POST`   | `/api/posts/:id/favorite` | Добавить в избранное/ убрать из избранного          |

### Поиск и медиа

| Метод  | Путь                | Описание                            |
| :----- | :------------------ | :---------------------------------- |
| `GET`  | `/api/search?q=...` | Поиск по имени, фамилии или `@тегу` |
| `POST` | `/api/upload/image` | Загрузка файлов на сервер           |
